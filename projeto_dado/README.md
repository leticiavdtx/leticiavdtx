Este projeto é um simulador de rolagem de dados feito em Python.
Ele permite que o usuário escolha quantos dados deseja rolar e qual tipo de dado usar (d4, d6, d8, d10, d12, d20 ou d100).

O programa também registra o histórico de rolagens e, a cada 20 rolagens, apresenta estatísticas básicas utilizando a biblioteca Pandas.

Este projeto foi desenvolvido como exercício de aprendizado de Python, modularização de código e criação de interfaces gráficas com Tkinter.

Funcionalidades

Simulação de rolagem de dados

 Suporte para múltiplos tipos de dados

 Estatísticas das rolagens a cada 20 execuções

Interface gráfica utilizando Tkinter

 Separação entre lógica do programa e interface

Tecnologias utilizadas

Python

Tkinter (interface gráfica)

Pandas (análise estatística)

Random (geração de números aleatórios)
